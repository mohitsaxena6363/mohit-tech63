HTML - HyperText Markup Language

HTML is the standard language used to create web pages. It defines the structure and content of a webpage.

<!DOCTYPE html>
<html>
  <head><title>Page Title</title></head>
  <body>
    <h1>Welcome to HTML</h1>
  </body>
</html>

CSS - Cascading Style Sheets

CSS is used to style and layout HTML elements. It controls colors, fonts, spacing, and more.

body {
  background-color: lightblue;
  font-family: Arial;
}
h1 {
  color: darkblue;
}

Python - Powerful Programming Language

Python is a high-level, interpreted programming language known for its readability and versatility.

def greet(name):
    print("Hello, " + name)

greet("Mohit")

Java - Object Oriented Language

Java is a versatile and platform-independent programming language used in Android, enterprise, and web development.

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
